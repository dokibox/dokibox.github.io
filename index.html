dokibox

~the lightweight music player for Mac OS X~

Download now View on GitHub
dokibox has not yet been released. Please come back later.

Simple, fast and lightweight

Handles libraries with tens of thousands of songs effortlessly, with much lower memory usage than behemoths like iTunes. GPU accelerated UI, too.

Free and open source

All good things are free, and in this case we mean free as in speech. Bugfixes, pull requests and other development welcome.

Supports many formats

Got a lot of songs in FLAC, Ogg Vorbis, MP3, AAC or Apple Lossless? All no problem, unlike iTunes.

Last.fm scrobbling

No need to download any plugins or extra programs. Scrobbling to last.fm works straight out of the box.

Extensible with plugins

Some obscure codec that dokibox doesn't support? Extend its capabilities with a plugin.

Designed solely for OS X

No cross-platform kludges allows dokibox to behave and feel just like how all OS X apps should.
